## Product phases

| Phase | Target | Scope |
| --- | --- | --- |
| Phase 1: Circle Testnet Integration | Current | Circle Programmable Wallet setup, testnet USDC wallet provisioning, sandbox API integration, balance sync, basic deposit/transfer flow, webhook handling, and Supabase transaction logging. |
| Phase 2: Testnet Payment MVP | Next | Hosted payment intent, merchant payment link, customer testnet checkout, payment status tracking, webhook verification, and basic merchant dashboard. |
| Phase 3: Merchant Testnet Tools | After MVP | Invoice create/send/pay flow, refund simulation, transaction history, API key management, webhook subscriptions, and developer quickstart docs. |
| Phase 4: Cross-Chain and Automation | Later | CCTP testnet bridge, auto-convert simulation, AI invoice reminder, fraud flagging, bridge route recommendation, and operational audit logs. |
| Phase 5: Public Testnet Launch | Final testnet milestone | Public testnet docs, faucet/setup guide, known limitations, sample apps, feedback form, and deployed contract/address registry if on-chain modules are added. |

## Future capabilities

| Feature | Description | Phase |
| --- | --- | --- |
| USDC yield | Idle USDC earns yield through Aave or Circle Yield-style integration. | 4 |
| QR POS payments | Physical store checkout by QR scan. | 4 |
| AI expense categories | Agent tags transactions for merchant accounting. | 3 |
| Subscription engine | Recurring billing with auto-charge for SaaS and subscription merchants. | 3 |
| White-label SDK | Fintech companies embed AbabilPay under their own brand. | 3 |
| Carbon offset donations | Optional micro-donation per transaction for ESG programs. | 4 |
| Loyalty points | Consumers earn fee-discount points for paying with AbabilPay. | 4 |
| AI price recommendations | Agent suggests invoice pricing based on market and customer patterns. | 4 |

## One-line pitch

AbabilPay is Stripe for the crypto economy: USDC-native, multichain, low-volatility for merchants, and powered by an AI agent layer on top of Supabase and Circle APIs.
