# AbabilPay

AbabilPay is a USDC-native payment infrastructure platform for consumers, merchants, developers, and enterprise teams. It abstracts chain complexity, keeps balances denominated in USDC, and automates payment operations through an AI agent layer.

<Note>
  Product docs version: v1.0 draft, May 2026. Core infrastructure: Circle APIs, Supabase, and AI agents.
</Note>

## Positioning

AbabilPay is built to be Stripe for the crypto economy: multichain-ready, USDC-first, low-volatility for merchants, and practical for real businesses that need checkout, invoices, payouts, and reporting.

Everything inside the product is denominated in USDC. Customers can deposit or pay with supported tokens, while AbabilPay handles conversion, bridge routing, wallet provisioning, and transaction records behind the scenes.

## Who it serves

| User type | Who they are | Primary goals | Key surfaces |
| --- | --- | --- | --- |
| Consumer | Individual users | Pay merchants, send or receive USDC, manage wallet activity | Wallet, bridge, split payments, on-ramp |
| Merchant | Stores, freelancers, SaaS teams | Accept payments, issue invoices, track revenue | Dashboard, invoices, plugins, off-ramp |
| Developer | App, Shopify, and WooCommerce builders | Integrate crypto payments into products | REST API, SDKs, webhooks, sandbox |
| Enterprise | Larger businesses and DAO treasuries | Bulk payments, payroll, approvals | Payroll, multi-sig, white-label API |
| AI agent system | Autonomous platform actor | Automate operational work | Agent orchestrator, Supabase, Circle tools |

## Platform stack

| Layer | Technology | Role |
| --- | --- | --- |
| Web frontend | Next.js 14, TypeScript | Merchant dashboard, consumer wallet, developer portal |
| Mobile | React Native with Expo | iOS and Android wallet experience |
| Backend API | Fastify and Node.js | REST API, orchestration, integrations |
| Data and auth | Supabase | PostgreSQL, Auth, Realtime, Storage, Edge Functions, RLS |
| Wallets and USDC | Circle APIs | Developer-controlled wallets, USDC, CCTP, ramps |
| Swaps | 1inch and 0x | Token-to-USDC conversion quotes and execution |
| AI | Claude API, LangChain, pgvector | Agent reasoning, orchestration, and long-term memory |
| Notifications | Resend, Firebase FCM | Transactional email and mobile push |
| Commerce | Shopify App, WordPress plugin | E-commerce checkout integrations |

## Why Supabase

Supabase is the primary application backend because it combines PostgreSQL for structured transaction data, Auth for Google OAuth, Realtime for live wallet and invoice updates, Storage for invoice PDFs and KYC documents, Edge Functions for event handling, and pgvector for AI agent memory. Row Level Security policies enforce user-scoped access across product data.
